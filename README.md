---

## 🔍 Overview  

Predicting churn is important — but **understanding and acting on it is what drives real business value.**

**CHurnSensePro** uses multiple machine learning models to generate a **churn prediction score** for each customer.  
The system helps identify:
- High-risk customers  
- Key churn drivers  
- Opportunities for targeted retention strategies  

---

## ✨ Key Features  

- 🧠 **Multi-Model Learning**  
  Logistic Regression, Random Forest, and XGBoost  

- ⚖️ **SMOTE Balancing**  
  Handles class imbalance for better churn detection  

- 📊 **Exploratory Data Analysis (EDA)**  
  Understand customer behavior and trends  

- 📈 **Model Evaluation Suite**  
  Confusion matrix, ROC curve, and performance metrics  

- 🔎 **Feature Importance Analysis**  
  Identify what drives customer churn  

- 🧩 **Modular Pipeline**  
  Clean and reusable code structure
